# Bundled report font

`NotoSansSC-Regular.ttf` is a fixed Regular-weight instance of Noto Sans SC, used by both charts and PDF rendering so Chinese
text is reproducible across Windows, Linux, CI, and reviewer machines.

The font metadata identifies the copyright holder as Adobe (2014-2021) with Reserved
Font Name "Source" and licenses the font under the SIL Open Font License 1.1. The full
license is included in `OFL.txt`.
